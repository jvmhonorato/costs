## Technology 

Here are the technologies used in this project.

* React JS
* CSS





## Services Used

* Github





## Getting started

* Dependency
- Json-server
- React-dom
- React-icon
- React-router-dom
 
  
  
* To install the dependencies.
```bash
$ npm install
```
  $ npm install
  
* To run the project.
```bash
$ npm run start
$ npm start
```
